Додаток дозволяє користувачу вводити текст помилки через форму на фронтенді, відправляти його на сервер для збереження та миттєво бачити оновлений список усіх записів у таблиці.

Функціонал:

Сервер: написаний на Express. Має два маршрути: GET (забрати всі логи) та POST (додати новий запис).

Валідація: сервер не пропустить порожній текст або коротше 3-х символів (поверне 400 помилку).

Фронт: на React + Vite. Використовуються хуки useState та useEffect.

Оновлення: коли додаєш новий запис, таблиця оновлюється сама, сторінку оновлювати не треба.

Стек:
Node.js, Express, Cors, React (Vite), CSS.

Як запустити:
Спочатку треба встановити залежності в обох папках.

1. Запуск сервера:
Bash
cd server
npm install
node index.js
(Буде на http://localhost:5000)

2. Запуск клієнта:
Bash
cd client
npm install
npm run dev
(Буде на http://localhost:5173)
